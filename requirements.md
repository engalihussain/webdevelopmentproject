# Web Development Project

## Project Description
Create a website about any selected topic.  
The website must fulfill the following requirements.

---

# HTML Requirements (2 Points)

The website must contain:

- Media:
  - Image
  - Video
  - Sound
  - (0.5 pts)

- Form
  - (0.5 pts)

- List or Table
  - Try to apply Spanning
  - (0.5 pts)

- Link to another webpage
  - (0.5 pts)

---

# CSS Requirements (2 Points)

- The website must have an external CSS document linked to it.
  - (1 pt)

- The CSS document must contain examples of:
  - HTML tag selector
  - ID selector
  - Class selector

- Design the web pages well using different CSS styles:
  - Background color
  - Background image
  - Font color
  - Font type
  - List styling
  - Table styling
  - Borders
  - etc.
  - (1 pt)

---

# JavaScript Requirements (2 Points)

The website must include dynamic behavior using JavaScript, such as:

- Events
- Change content
- Functions
- etc.

---

# PHP & MySQL Requirements (2 Points)

Use PHP to connect with MySQL database.

Requirements:

- Use HTML Forms and PHP
- Get and set information between Database and Webpage
- Apply operations such as:
  - Insert
  - Select
  - Delete
  - Update
  - etc.

---

# Presentation & Discussion (2 Points)

- Project presentation
- Discussion

---

# Total Marks

- Total: 10 Points
